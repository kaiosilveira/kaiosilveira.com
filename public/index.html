Freelances

"Those who can imagine anything, can create the impossible.”

— Alan Turing

Organização

Costumo utilizar o Trello para centralizar o desenvolvimento dos projetos que assumo. Crio um board, particiono o escopo em tarefas e utilizo um KANBAN simples (Todo → In Progress → Validation → Done) para me organizar. Forneço acesso às partes envolvidas / interessadas no projeto para termos um progresso claro e transparente das atividades que estão sendo realizadas e de todas as tarefas que devem ser feitas.

Documentação

Utilizo o Google Drive para centralizar toda e qualquer documentação. Nele, geralmente crio uma pasta com o nome do projeto ou da empresa (em caso de múltiplos projetos) e forneço acesso às partes envolvidas/interessadas, para que a documentação se mantenha viva (atualizada tanto por mim quanto pelos envolvidos), transparente e atualizada.
